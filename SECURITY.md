# Security Policy

## Supported Versions

Currently, only the latest version of this plugin is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability in the Princi plugin, please do not open a public issue.

Instead, please report it by emailing **security@princi.ai**.

### What to expect

- You will receive an acknowledgment of your report within 48 hours.
- We will provide a timeline for a fix and keep you updated on our progress.
- Once the issue is resolved, we will post an update to the repository.
